This README explains the structure of the documentation folders for 
the **purl-spec** project. See the README files in the sub-folders for more
detailed information about the content of a sub-folder.

The `docs` sub-folders are:
- `decisions`: Decision documents for the PURL Specification including 
information like context, decision drivers, and alternatives considered.
- `drafts`: Draft versions of major documents.
- `misc`: Miscellaneous documents.
- `specification`: Documents for the PURL Specification with the subset of 
documents that comprise ECMA-427 in the `standard/` sub-folder.
- `types`: Documents about PURL **types**. The `definitions/` sub-folder
contains documentation generated from the **type** definition JSON files. 
The `notes/` sub-folder contains information about **types** such as the 
reasons for creating a **type** or guidance about the usage of a **type**.
